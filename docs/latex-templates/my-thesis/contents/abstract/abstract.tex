\textit{
This research aims to design and implement a blockchain-based cooperative system using Decentralized Autonomous Organization (DAO) principles. The developed system, named Lakomi (Digital Cooperative Service), implements core cooperative functions including treasury management, democratic governance through voting, collateral-backed emergency loans, and contribution-based tiered lending as smart contracts on the blockchain. The methodology employed is Design Science Research (DSR) with stages of problem identification, architecture design, smart contract implementation using Solidity and the Foundry framework, deployment on the DChain network, and security and efficiency evaluation. The results demonstrate that the Lakomi system successfully implements four smart contracts (LakomiToken, LakomiVault, LakomiGovern, and LakomiLoans) with 105 unit tests all passing. The system enhances cooperative management transparency through immutable transaction records and decentralized voting mechanisms. The conclusion of this research is that DAO implementation on the DChain network can serve as a viable alternative solution for modernizing the Indonesian cooperative model.
}

\noindent\textbf{Keywords} : blockchain, cooperative, DAO, smart contract, governance, DChain
